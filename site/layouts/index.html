Installation

  • Download the binary for your platform here and place it somewhere on your $PATH
  • Run eval "$(leader init)" to activate leader.
  • Paste the example configuration into ~/.leaderrc
  • Press \ in your terminal to trigger leader.
  • If you like it, add eval "$(leader init)" to your shell's initialization file (e.g ~/.bashrc or ~/.zshrc)

Configuration

Place the following in ~/.leaderrc to get started. 

{
  "keys": {
    "h": {
      "name": "help",
      "keys": {
        "l": "leader help"
      }
    },
    "g": {
      "name": "git",
      "loopingKeys": [
        "s"
      ],
      "keys": {
        "p": "git push",
        "P": "git pull",
        "s": "git status"
      }
    }
  }
}
